При первом запуске программа сама создает нужную БД, нужно только в файле
src/resources/application.properties выставить "spring.jpa.hibernate.ddl-auto=update"
а в строчках spring.datasource.url=jdbc:mysql://localhost:3306/quizdb
             spring.datasource.username=***
             spring.datasource.password=***
ввести данные своей БД

localhost:8080/newQuiz -страница для создания нового опроса.
посла создания сгенерируются 3 ID-для прохождения опроса, его удаления и просмотра статистики.
Сделано для того, чтобы можно было проходить опрос без регистрации.
Соответственно на localhost:8080/quizPage вводим ID опроса- голосуем
На localhost:8080/deletePage вводим ID опроса на удаление-опрос стирается
На localhost:8080/statisticsPage вводим ID опроса для просмотра статистики-получаем информацию


Данное разделение на 3 ID вместо одного сделано для того, чтобы кто угодно создать опрос
без регистрации,и, выдавая ID,позволял бы людям его проходить, и в то же время только имеющий конкретные
ID мог узнать результат и удалить опрос.
